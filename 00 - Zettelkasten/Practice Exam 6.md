CSR
- a cryptographic file generated by an entity requesting a digital certificate from a CA

Wildcard certificate
- digital certificate that can be used to secure multiple subdomains within a primary domain

Subject Alternative Name (SAN)
- allows to secure multiple domain names or subdomains with a single certificate

OID
- identifier used for PKI objects

Agent-less software threat vectors
- Network protocol vulnerability
- Packet sniffing
	- involve security risks that exploit vulnerabilities, misconfigurations, or API dependencies without requiring installed, persistent agents on target systems
