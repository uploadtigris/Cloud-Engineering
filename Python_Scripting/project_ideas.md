# Project Ideas

1. System Health Monitor

Check CPU, memory, disk usage
Monitor running processes
Alert when thresholds are exceeded
Uses: psutil library, cron jobs

2. Log Rotation & Cleanup

Find and compress old log files
Delete logs older than X days
Generate summary reports
Learn: file operations, os and pathlib modules

3. User Account Manager

Create bulk users from CSV
Set up SSH keys automatically
Audit user permissions
Learn: subprocess module, interacting with Linux commands

4. Service Health Checker

Monitor systemd services
Restart failed services automatically
Send alerts via email/Slack
Learn: systemd interaction, daemon processes

5. Backup Automation

Automated backup of critical directories
Compress and transfer to remote server
Verify backup integrity
Learn: tarfile, paramiko (SSH), scheduling

6. Network Port Scanner

Check which ports are open on local/remote systems
Identify running services
Compare against baseline
Learn: socket module, network basics
