# ☁️ Phase 2: Cloud Infrastructure
**Goal:** Learn to deploy, manage, and automate infrastructure in AWS.

---

## ✅ Projects Checklist

### Complete AWS Certified Cloud Practitioner (CCP)
- [ ] Study AWS core services (EC2, S3, RDS, VPC, IAM)
- [ ] Understand AWS pricing models and billing
- [ ] Learn AWS security and compliance fundamentals
- [ ] Take practice exams and review weak areas
- [ ] Schedule and pass AWS CCP certification

---

### Progress to AWS SysOps Administrator and AWS Networking specializations
- [ ] Deploy and manage EC2 instances (launch, connect, configure)
- [ ] Configure VPCs with subnets, route tables, and security groups
- [ ] Set up IAM users, roles, and policies for secure access
- [ ] Practice monitoring with CloudWatch (metrics, logs, alarms)
- [ ] Work with Load Balancers and Auto Scaling groups
- [ ] Study for AWS SysOps Administrator certification

---

### Apply Linux knowledge to deploy secure, scalable cloud environments
- [ ] Launch Rocky Linux EC2 instances
- [ ] Configure SSH key pairs and security groups
- [ ] Deploy web servers on EC2 with proper security hardening
- [ ] Set up RDS databases and connect to applications
- [ ] Use S3 for backups and static content storage
- [ ] Implement CloudWatch monitoring for Linux instances

---

### Automate provisioning with CloudFormation or Terraform
- [ ] Learn Infrastructure-as-Code (IaC) concepts
- [ ] Write CloudFormation templates for basic infrastructure (VPC, EC2, S3)
- [ ] Install and configure Terraform
- [ ] Create Terraform configurations for multi-resource deployments
- [ ] Practice terraform init, plan, apply, and destroy
- [ ] Version control IaC templates in GitHub

---

**Skills Learned:** AWS core services · Infrastructure-as-Code (IaC) · Security · Monitoring · Automation workflows
