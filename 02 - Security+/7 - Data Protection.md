Data Types
- TLS & SSL
	- cyptographic protocols
- VPN
	- secure connection over a less secure connection
- IPSec
	- protocol suite
	- authenticating and encrypting each IP packet in a data stream
- Data in use
	- Created, Update, Read, Deleted data in progress
- Protecting Health Information (PHI)
	- regulated by HIPPA
- Trade secrets
	- protected by law
- Intellectual Property (IP)
- Legal Information
- Financial Information
	- PCI DSS (payment card information data security standard)
- Non-human Readable Data

Data Sovereignty
- digital information subject to laws where it is located
- GDPR (general data protection regulation)

Securing Data
- geofencing - geographic restrictions
- encryption
	- plaintext to ciphertext with an encryption key
- Hashing
	- converts data into a hash value
- Masking
	- replacing data with an "x"
- Tokenization
	- replaces sensitive data with non-sensitive substitutes
- Obfuscation
	- scrambling data
- Segmentation
	- dividing network
- permission restrictions

Data Loss Prevention (DLP)
- protect data in use, transit, or at rest
- software
	- Endpoint DLP System
		- tracks the movement of data on a system
	- Network DLP System
		- on the network
	- Storage DLP
		- software on server in datacenters
	- Cloud-based DLP System
